📖 Help

            
Welcome to the Easy Bitmap Maker! Here's how to use the tool:

            
    
                
  • 🪄 Create Grid: Set the width and height (up to 128) and click to generate the grid
    • 
                
  • 🖌️ Fill Mode: LMB draws pixels, RMB erases pixels
    • 
                
  • 🧹 Erase Mode: LMB erases pixels, RMB draws pixels
    • 
                
  • 🎨 Pen Size: Adjust the pen size using the slider for larger or smaller brush strokes
    • 
                
  • 📥 Import Bitmap: Click to import a bitmap from C++ code
    • 
                
  • 💾 Generate C++ Code: Click to generate and view the C++ byte array representing your bitmap
    • 
                
  • 📋 Copy Code: Click to copy the generated C++ code to your clipboard
    • 
                
  • 🔄 Invert Colors: Click to invert the colors of all pixels
    • 
                
  • 🗑️ Clear Grid: Click to clear all pixels in the grid
    • 
                
  • 🔑 Hotkeys: Press D for Fill mode and E for Erase mode
    • 
                
  • ❓ Help: Click the Help button to view these instructions
    • 
                
  • 🌐 Language: Toggle between English and Russian using the button in the top-right corner
    • 
                
  • ⭐ Support: If you like this project, please star it on GitHub
    • 
            

            
Happy creating!
